## Setup
1. Install dependencies:
   ```bash
   npm install
   ```

2. Run tests:
   ```bash
   npm test
   ```

3. Run tests in watch mode:
   ```bash
   npm run test:watch
   ```

## Problem Notes
- You start at any position in the energy array
- You can only move forward by exactly k positions
- Goal: Find the maximum energy you can collect
